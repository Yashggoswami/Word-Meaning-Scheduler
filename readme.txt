Word Meaning Scheduler

This is a simple Application that helps user to maintain habit of learning new words each day. The user simply input the Number of words he/she wants to learn in a day and select days of the week. Then the user receives word-meaning notifications on there system on regular time interval. This Application reduced the headache of searching new words each day and hence the user just need to look at the notification to learn a new word-meaning.

Python modules used - 

1. plyer
2. schedule - https://pypi.org/project/schedule/
3. pydictionary
